# Setup memorydecay Plugin

> Learned skill -- auto-generated by memento-skills
> Utility: -- (0/0) | Last optimized: never

## When to Use
- User runs `/plugin marketplace add` and it fails
- User asks to install, setup, or configure this plugin
- New contributor cloning this repo for the first time

## Procedure
1. **Clone memory-decay-core** (required dependency):
   - Check if `../memory-decay-core/` already exists
   - If not, clone it: `git clone https://github.com/memory-decay/memory-decay-core.git ../memory-decay-core`
   - Set up its venv: `cd ../memory-decay-core && uv venv && uv pip install -e .`
   - Record the absolute path for step 6
2. Install the Python package via `uv tool install --from . claude-code-memorydecay` (or `uv venv` + `uv pip install -e .` as fallback)
3. Create directories: `~/.claude/skills`, `~/.claude/hooks`, `~/.memorydecay`
4. Copy skill: `cp -r .claude/skills/memorydecay ~/.claude/skills/`
5. Copy hooks: `cp .claude/hooks/pre-compact .claude/hooks/session-end ~/.claude/hooks/` and `chmod +x` them
6. Add `export MEMORYDECAY_CORE_PATH=<absolute-path-to-memory-decay-core>` to `~/.zshrc` (or user's shell profile)
7. Verify with `memorydecay --version` and `memorydecay server status`

## Known Failure Modes
- `uv pip install --system` fails on externally-managed Python environments — use `uv tool install` or a venv instead
- Marketplace install fails if `.claude-plugin/marketplace.json` is missing (current state of this repo)

## Examples
```bash
# Typical successful install sequence
# 1. Clone the core dependency and set up its venv
git clone https://github.com/memory-decay/memory-decay-core.git ../memory-decay-core
cd ../memory-decay-core && uv venv && uv pip install -e .
cd -

# 2. Install the CLI
uv tool install --from . claude-code-memorydecay

# 3. Install skill and hooks
mkdir -p ~/.claude/skills ~/.claude/hooks ~/.memorydecay
cp -r .claude/skills/memorydecay ~/.claude/skills/
cp .claude/hooks/pre-compact .claude/hooks/session-end ~/.claude/hooks/
chmod +x ~/.claude/hooks/pre-compact ~/.claude/hooks/session-end

# 4. Set environment variable
export MEMORYDECAY_CORE_PATH=$(realpath ../memory-decay-core)

# 5. Verify
memorydecay --version  # → memorydecay, version 0.1.0
```
